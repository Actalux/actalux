How Actalux Works

What we publish

Actalux indexes official Clayton School District documents: board meeting agendas, minutes, board packets, and resolutions. Every search result is a verbatim quote from an official document.

Document processing

Documents are converted from their original format (PDF or HTML) to plain text. The text is split into passages of approximately 200 words, preserving section boundaries and sentence integrity. No text is paraphrased or rewritten.

How search works

Search uses two methods simultaneously:

  • Keyword matching finds passages containing your exact search terms.
  • Semantic search finds passages with similar meaning, even if the exact words differ.

Results from both methods are combined using reciprocal rank fusion, which weights passages that appear in both result sets higher than those appearing in only one.

Citation integrity

At ingestion time, every passage is verified to be an exact substring of its source document. Passages that fail this check are rejected. Each displayed passage includes a short hash ID (e.g., #q003f) that serves as a stable, shareable reference.

What we do not do

  • We do not editorialize, summarize without citation, or express opinions.
  • We do not publish closed session content.
  • We do not publish content about individual personnel, teachers, or students.
  • We do not infer intent or make causal claims unless explicitly stated in the source document.

Corrections

If you find an error, use the "Report an error" form on any page. Include the quote ID so we can locate the passage. We review every report.

Open source

The full source code is available at github.com/Actalux/actalux. Actalux is a 501(c)(3) nonprofit organization.
